# Ionic Lib
Utils Ionic library in Typescript

Some Common components, services and util functions.

## Installation

### Install from git
```bash
mkdir -p lib
cd lib
git submodule add https://github.com/beenotung/ioniclib.git
```

### Install from npm
```bash
npm install ioniclib
```

## Remark
 - Target Ionic 3+, Angular 4+
